Максимально спрощене домашнє завдання із використанням api mysqli.
===
Необхідно відобразити інформацію із однієї таблиці бази даних, яку проектувала ваша команда.
Для відображення інформації необхідно використовувати api mysqli
Див. детальніше за посиланням: https://www.php.net/manual/ru/book.mysqli.php
Обрана таблиця не повинна повторюватися в середині команди.
Лише один php файл.
